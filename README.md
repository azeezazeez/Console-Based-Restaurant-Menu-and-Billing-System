# 🍽️ Restaurant Menu & Billing System (Java)

A simple console-based restaurant ordering and billing system built using **Core Java**.
This application allows users to select food items from different categories and calculates the total bill based on quantity.

---

## 🚀 Features

* 📋 Menu-driven interface
* 🍗 Multiple food categories:

  * Starters
  * Main Course
  * Desserts
* 🔢 Quantity-based billing
* ⚡ Instant total calculation
* 🔁 Continuous ordering system (loop-based)
* ❌ Handles invalid inputs

---

## 🛠️ Tech Stack

* **Language:** Java (J2SE)
* **Concepts Used:**

  * Switch-case
  * Loops (for continuous execution)
  * Scanner class (user input)
  * Basic OOP (class & methods)

---

## 📂 Project Structure

```
Restaurant1.java
```

---

## 📜 How It Works

1. User runs the program
2. Menu categories are displayed:

   * Starters
   * Main Course
   * Desserts
3. User selects a category
4. Chooses an item from the list
5. Enters quantity
6. Total amount is calculated and displayed

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/azeezazeez/restaurant-billing-system.git
```

2. Compile the program:

```
javac Restaurant1.java
```

3. Run the program:

```
java Restaurant1
```
